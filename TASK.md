# 장사왕 재고관리 MVP 목업 — 구현 태스크

## 목표
Next.js 14 + Tailwind CSS + shadcn/ui 기반으로 장사왕 재고관리 MVP 화면을 빠르게 구현.
실제 기능보다는 화면 구성 검증 및 시각화가 목적.

## 기술 스택
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui
- lucide-react 아이콘
- Mock 데이터 (실제 API 없음)

## 구현할 화면 4개

### 1. /dashboard — 대시보드 캘린더
- 월별 캘린더
- 날짜 셀: 입고권장(클릭링크), 재고주문(표시), 입고중(클릭링크), 매출, 순이익
- 쇼핑몰 필터 (전체/로켓/그로스/윙 탭)
- 입고권장 클릭 → /inventory/orders?tab=recommend
- 입고중 클릭 → /inventory/orders?tab=incoming

### 2. /products — 상품관리
- 테이블 컬럼: + | 별칭 | 상품명 | 옵션명 | 판매가 | 원가(이력) | 배송비 | 마진 | 마진율 | ROAS | 리뷰수 | 리뷰점수 | 마켓재고 | 국내총재고 | 수정일
- + 클릭 → 원가 이력 행 토글 (날짜별 원가 목록)
- 마켓재고 클릭 → Dialog (추가/수정 라디오, 수량 입력)
- 우측상단 [바코드 인쇄] 버튼 → Dialog (용지선택, 상품체크, 수량, PDF/인쇄)
- 마진 = 판매가 - 원가 - 배송비 자동계산
- 마진율 = 마진/판매가 * 100
- ROAS = 판매가/마진 * 100

### 3. /inventory/daily — 재고관리(일일매출/재고)
- 상단: 날짜 표시, 조회일 기준
- 테이블: + | 상품명 | 순이익금 | 재고금액 | 월판매수량 | 일판매수량 | 마켓재고 | 국내총재고 | 일목표수량(편집가능) | 입고권장 | 입고중 | 재고입고 | 재고주문
- 입고권장: 계산값 배지 표시 (빨강경고/초록충분)
- 재고주문 [발주] 버튼 → Dialog (발주수량, 구매단가, 총액자동계산, 메모)
- + 클릭 → 옵션별 행 펼치기

### 4. /inventory/orders — 재고관리(주문)
- 탭 9개: 입고권장 | 발주요청(disabled) | 발주금액확정(disabled) | 구매확정 | 구매완료 | 중국창고도착 | 중국창고출고 | 한국입항(입고중) | 상품수령완료 | 전체
- 공통 테이블: 체크박스 | NO | 구매요청일 | 상품명 | 발주량 | 단가 | 총액 | 현재상태 | 공장출고예정일 | 재고도착예정일 | 트래킹번호 | 관부가세 | 국내운임비 | 메모
- 상단 액션: [이동→] [주문취소] 버튼
- 상품수령완료 탭: [입고처리] 버튼 → Dialog (실수량, 입고일)
