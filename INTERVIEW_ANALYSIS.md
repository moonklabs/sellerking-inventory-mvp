# 고객 인터뷰 분석 - AS Gel 전인수 대표님 (2026.02.13)

## 핵심 인사이트

### 페인포인트
- 재고가 가장 큰 스트레스 → 품절 = 순위 하락 = 매출 손실
- 기존 솔루션(트렌드온, 빅셀, 셀로, 엑셀) 모두 재고는 불완전
- 엑셀 수동 작업 너무 많고 힘듦
- 쿠팡 그로스 재고 API 없어서 스크래핑만 가능

### 핵심 니즈 (한 문장)
> "입력 최소화, 보기 최대화. 재고 몇 개 있고, 발주 언제 넣어야 하는지만 보여주면 된다"

## 필수 기능 (Must Have)

| # | 기능 | 근거 발언 |
|---|------|----------|
| 1 | 쿠팡 그로스 마켓재고 자동 수집 (스크래핑) | "거기서 자동으로 끌고 올 거고" |
| 2 | 국내 총재고 수동 입력 (마켓재고와 분리) | "마켓 재고를 따로 잡았어요" |

| 4 | 입고권장 수량 자동 계산 | "월 목표 수량 기준으로 몇 개 입고하면 좋겠다 알려주면 편해" |
| 5 | 원가 이력 관리 (날짜별 입고원가 기록) | "플러스 누르면 몇 월에 얼마에 들어왔고 보여주면 좋겠다" |
| 6 | 바코드 인쇄 | "그로스 입고 시 바코드 출력 엄청 힘들어요" |
| 7 | 마진율 / ROAS 표시 | "ROAS 뷰 밑으로 들어가면 안 되니까 그것만 보여주면" |
| 8 | 발주 9단계 프로세스 | "체크 누르고 이동 누르면 다음 프로세스로" |

## POC 추가 화면 정의

### 추가 화면 1: 재고 현황 대시보드 (실시간)
- 국내 총재고 vs 마켓재고 비교 뷰
- 입고권장 알림 배너
- 상품별 재고 위험도 표시 (빨강/노랑/초록)

### 추가 화면 2: 입고 이력 관리
- 상품별 입고 날짜 / 수량 / 원가 타임라인
- 플러스 버튼으로 신규 입고 등록
- 입고 시 국내 총재고 자동 반영

### 추가 화면 3: 월 목표 설정
- 상품별 월 목표 수량 설정
- 시즌 상품 표시 (시즌 온/오프 토글)
- 목표 기반 입고권장 자동 재계산 미리보기

### 추가 화면 4: LCL 원가 배분 계산기
- 여러 상품 한 번에 입고 시 관부가세/배송비 자동 배분
- 배분 방식 선택: N분의 1 / 가격 비례 / 무게 비례
- 배분 결과 원가에 자동 반영

### 추가 화면 5: 마켓별 재고 입력
- 그로스 / 로켓 각각 발송 수량 입력
- 국내 총재고에서 자동 차감
- 발송 이력 타임라인
